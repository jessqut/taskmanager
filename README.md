Personal Workout Tracker
This project is for a personal workout tracker app that provides users with the option to track their workouts and monitor their progress.
Features include:
•	Create and edit personal accounts
•	Log in and log out of personal accounts
•	Create, edit and delete workouts
•	Record workout title, details, date, calories burned
•	Display user’s current workout progress
•	Display user’s workout history
•	User friendly application, easy to navigate and reliable progress tracking
However, to date, just the basic functions have been completed – Sign up and Log in.

To help create the app and to visualise the system requirements, 9 SysML diagrams are drawn using Drawio. These diagrams can be found in the frontend/SysML_diagram path.
For project management, JIRA was used. The main tasks were outlined with the sub-tasks detailing the intended processes.

The UI/UX designs were designed in Figma and can be found in frontend/ui-ux path.

How to run:
This project is designed to run in a Linux Environment such as ubuntu. It will not work if the commands are run through the desktop terminal.
For Windows users, you can connect to the Linux Environment using PuTTY and run it there.
To run the backend: node server.js
To run the frontend: npm start
